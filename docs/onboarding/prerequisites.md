# Prerequisites

Before you begin, ensure you have the following installed:

## Required Software

*   **Python**: Required for running the agent server.
*   **Uv**: A fast Python package installer and resolver.
*   **Git**: For cloning the repository.

## API Keys

You will need a Gemini API key to run the agents.

*   **Get an API Key**: Visit [Google AI Studio](http://aistudio.google.com/) and click "Get API Key" in the top left corner.

## Google Cloud Setup (Optional but Recommended)

If you plan to use Vertex AI or deploy your agents:

1.  **Install gcloud CLI**: Make sure `gcloud` is in your PATH.
2.  **Initialize gcloud**: Run `gcloud init` and select a project.
3.  **Billing**: Ensure billing is enabled for your Google Cloud project.
